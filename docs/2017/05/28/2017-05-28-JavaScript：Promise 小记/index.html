JavaScript：Promise 小记

发表于 2017-05-28 | 评论：

Promise 对象是一个代理对象（代理一个值），被代理的值在 Promise 对象创建时可能是未知的。它最主要的特点，是可以让异步代码像同步方法那样返回值，并为结果的成功（onFulfilled）和失败（onRejected）绑定相应的处理方法（Handlers）。需要注意的是，尽管看起来像同步方法那样返回了值但是并不是立即返回执行的。

一个 Promise 有以下几种状态：

  • pendding 初始状态
  • fulfilled 意味着操作成功
  • rejected 意味着操作失败
