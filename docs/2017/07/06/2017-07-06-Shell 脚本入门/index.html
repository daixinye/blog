二、编写的三个要素

观察一下我们的 helloworld.sh，我们可以把它分为三部分：指定程序、注释说明 以及 脚本命令。

1、指定程序：#!/bin/sh

符号 #! 用于指定运行该脚本使用的程序。

在 helloworld.sh 中，我们用 /bin/sh来执行这个脚本。

2、注释说明：# it's just a demo

注释，以 # 开头的一行表示注释。

3、脚本命令： echo "Hello World"

echo用于在命令行中打印指定的字符串。

你可以用单引号、双引号将要打印的字符串包裹起来，也可以选择不用引号。

三、运行的两个方式
