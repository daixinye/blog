水平居中

1、宽度已知，使用margin

2、宽度不定，使用text-align + inline-block

3、宽度不定，使用弹性盒子

4、宽度已知，使用绝对定位

完全居中

1、完全居中

2、使用负margin

优点：兼容IE6-7；

缺点：不能使用百分比的大小，内容高度不可变；内容可能会超出容器；

3、使用transform

优点：内容高度可变；

缺点：不兼容IE8；会和其他transform样式有冲突；

4、使用table-cell

优点：内容高度可变；能自动撑开父元素；浏览器兼容性好；

缺点：需要额外的HTML标签；

5、使用flex-box

优点：内容高度可变；

缺点：不支持IE8-9；
