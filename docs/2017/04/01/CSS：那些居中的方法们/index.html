2、使用负margin

优点：兼容IE6-7；

缺点：不能使用百分比的大小，内容高度不可变；内容可能会超出容器；

3、使用transform

优点：内容高度可变；

缺点：不兼容IE8；会和其他transform样式有冲突；

4、使用table-cell

优点：内容高度可变；能自动撑开父元素；浏览器兼容性好；

缺点：需要额外的HTML标签；

5、使用flex-box

优点：内容高度可变；

缺点：不支持IE8-9；
